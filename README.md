### Git Branch Rule
- main: For release (Don't touch)
- develop: Most recent updated branch
- branch naming: ```[ISSUETYPE]-[ISSUENUMBER]/[park or kim]/[ISSUENAME]```
  - ex1) enhancement-17/park/kakao-oauth
  - ex2) refactoring-12/kim/refactor-controller


### Commit message form

```"[TPYE] - [message]"```
